
Spyderwebb
==================

JWST NIRspec multi-object spectroscopy reduction software.

Authors
-------

- **David Nidever** (Montana State University)
  
Installation
------------

To install the latest version::

    pip install git+https://github.com/dnidever/spyderwebb
